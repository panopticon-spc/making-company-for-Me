수어 연습 가이드

소개
이 프로젝트는 카메라를 이용해 사용자의 손모양을 실시간으로 인식하고, 이를 기반으로 알맞은 수어를 화면에 텍스트나 그래픽으로 출력해 주는 수어 학습 및 연습 도구입니다.

주요 기능

 실시간 손모양 인식: 웹캠이나 카메라를 통해 사용자의 손동작을 실시간으로 감지합니다.
 수어 번역 및 출력: 인식된 손동작이 어떤 수어인지 분석하여 화면에 즉각적으로 매칭되는 단어나 문장을 출력합니다.
 수어 학습/연습 가이드: 사용자가 올바른 수어 동작을 따라 할 수 있도록 시각적인 피드백을 제공합니다.
 
사용 방법
1. 다운로드: 본 저장소(Repository)를 클론(Clone)하거나 ZIP 파일로 다운로드합니다.
2. 실행: 프로젝트 폴더에서 필요한 라이브러리를 설치한 후, 메인 스크립트(예: ⁠main.py⁠ 또는 ⁠npm start⁠)를 실행하고 카메라 권한을 허용합니다.
